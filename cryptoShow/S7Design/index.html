Ideja Šta donosi
Razdvajanje problema Smanjiti međusobnu zavisnost, a povećati povezanost (separation of concerns)
Povećavanje povezanosti Koriste se komponente, a ne šabloni (templates) (reduce coupling, increase cohesion)
Ponovna upotreba !!! (reusibility)

Zašto sam zavoleo React i JSX?

Omogućava da dizajner i developer u isto vreme rade na jednoj komponenti

JSX je opcioni pre-processor koji omogućava korišćenje HTML-like koda

Neophodni preduslovi za realizaciju projekta cryptoShow
instalirajte Visual Studio Code
nodeJS >= 6 i npm >= 5.2, proverite sa `npm --version`
mkdir cryptoShow && npx create-react-app cryptoShow
cd cryptoShow && npm install && npm start
rezultat vašeg rada se vidi u browser-u na URL http://localhost:3000
uštedećete vreme debug-ujući ako instalirate `React Developer Tools` addOn/extension u browser
startujte VSCode otvorite folder cryptoShow
uklonite celokupan sadržaj `src/` foldera ali ne i folder
kreirajte index.css i index.js fajlove unutar src/ foldera

Zapamtite, sve ovo će biti traćenje vremena ako se ne posvetite ovom projektu
