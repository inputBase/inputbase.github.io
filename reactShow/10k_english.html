crypto-show

CoinMarketCap API

crypto-show

feature #1 - choice of fiat currency
enable currency selection from the drop-down menu (USD, EUR, CNY)
the fiat currency dropdown must be available on all screens
a change on one of the screens is reflected on the others

crypto-show

feature #2 - list of cryptocurrencies
display a list of the 100 most popular cryptocurrencies
for each currency from the list display
rank, symbol, price in selected fiat currency and price change in the last 24 hours
filter/search for each parameter

button for refresh and auto-refresh lists after a certain time interval

crypto-show

Necessary prerequisites for the implementation of the crypto-show project
install Visual Studio Code
nodeJS >= 6 i npm >= 5.2, proverite sa `npm --version`
mkdir crypto-show && npx create-react-app crypto-show
cd crypto-show && npm install && npm start
the result of your work can be seen in the browser at URL http://localhost:3000
you will save time debugging if you install `React Developer Tools` addOn/extension in browser
start VSCode and open folder crypto-show
remove the entire contents of the `src/` folder but not the folder itself
create index.css and index.js files inside the src/ folder
